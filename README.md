# Luxury Watch Market Analytics Platform

## Project Overview
End-to-end analytics solution analyzing 500+ luxury watch models to provide 
market insights for collectors and retailers. Demonstrates data engineering, 
dashboard development, and business intelligence capabilities.

## Business Problem
- Need for data-driven pricing insights in luxury watch market
- Lack of standardized data quality metrics
- Multiple stakeholder needs (executives vs. detailed analysts)

## Technical Implementation

### Data Engineering
- Cleaned and standardized 500+ watch records
- Feature engineering: box/paper availability, case diameter, item condition, watch model imputation from description
- Created brand segmentation and year-based market segments
- SQL database design for efficient querying

### Analytics & Insights
- Feature-based pricing analysis
- Brand-condition-price relationship analysis
- Data quality assessment and monitoring

### Visualization & Dashboards (Tableau)
1. **Executive Summary Dashboard:** Inventory KPIs for management
2. **Data Features Completeness:** Highlights missing data for crucial features
3. **InventoryHealth:** Highlights suspicious listings and variety of model listings

## Technologies Used
- **Database**: PostgreSQL
- **Analysis**: Python (Pandas, NumPy)
- **Visualization**: Tableau, Matplotlib
- **Version Control**: Git

## Key Deliverables
- 3 interactive Tableau dashboards
- SQL database schema
- Automated data cleaning pipeline
- Data quality monitoring framework

## Business Impact
- Identified key pricing drivers in luxury watch market
- Enabled data-driven inventory decisions
- Established data quality baseline for reliable reporting
